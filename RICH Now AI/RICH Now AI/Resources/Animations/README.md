# 迎賓動畫整合指南

## 概述
本資料夾包含加百列財務守護天使的迎賓動畫資源。

## 動畫規格
- **時長**：8 秒
- **解析度**：1080x1920（手機直式螢幕）
- **幀率**：60fps
- **格式**：Lottie JSON
- **風格**：現代化、精緻、有質感
- **色彩**：金色、白色、深藍色

## 從 Gemini 獲取動畫

### 1. 使用提供的描述句
將以下描述句提供給 Gemini：

```
請製作一個令人驚艷的 8 秒迎賓動畫，描述如下：

【場景設定】
- 背景：深邃的星空，閃爍著金色和藍色的星光
- 氛圍：神秘而溫暖，充滿希望和力量

【動畫流程（總時長：8秒）】
第 0-1 秒：開場
- 螢幕從黑色漸變到星空背景
- 星星緩緩閃爍，營造神秘氛圍

第 1-2 秒：光環出現
- 一道金色光環從螢幕中央緩緩展開
- 光環散發溫暖光芒，逐漸變大

第 2-3 秒：羽毛飄落
- 幾片純白羽毛從上方優雅飄落
- 象徵天使即將降臨

第 3-5 秒：加百列出場
- 一位威嚴而慈祥的天使長從光環中緩緩升起
- 身穿白色長袍，金色腰帶
- 背後展開巨大的白色羽翼
- 手持金色權杖，頂端有發光的十字架
- 表情溫和而堅定

第 5-6 秒：文字浮現
- 「我是加百列」文字以金色光芒浮現
- 「你的財務守護天使」副標題緊隨其後

第 6-7.5 秒：光效高潮
- 天使周圍散發強烈金色光芒
- 光芒照亮整個螢幕
- 羽毛持續飄落

第 7.5-8 秒：結尾
- 天使微笑點頭，做出祝福手勢
- 光芒逐漸柔和，定格在溫暖畫面

【技術要求】
- 動畫時長：精確 8 秒
- 解析度：1080x1920（適合手機直式螢幕）
- 幀率：60fps（流暢動畫）
- 風格：現代化、精緻、有質感
- 色彩：以金色、白色、深藍色為主
- 特效：柔和的光效、羽毛飄落、星光閃爍
- 格式：MP4 或 MOV，支援透明背景

【情感表達】
- 威嚴中帶著慈愛
- 專業中帶著溫暖
- 給人安全感和信任感
- 體現財務守護者的專業形象
```

### 2. 轉換為 Lottie 格式
1. 使用 After Effects 或其他動畫軟體打開 Gemini 生成的動畫
2. 安裝 Bodymovin 插件
3. 導出為 Lottie JSON 格式
4. 確保動畫時長為 8 秒（480 幀 @ 60fps）

### 3. 替換佔位文件
1. 將生成的 `gabriel_welcome.json` 文件替換現有的佔位文件
2. 確保文件名稱為 `gabriel_welcome.json`
3. 測試動畫在 iOS 上的播放效果

## 技術實現

### Lottie 依賴
確保在 Xcode 專案中添加了 Lottie Swift Package：
- URL: https://github.com/airbnb/lottie-ios
- 版本: 最新穩定版本

### 動畫播放
動畫通過 `WelcomeAnimationView` 播放：
```swift
LottieView(animation: .named("gabriel_welcome"))
    .playing(loopMode: .playOnce)
    .animationSpeed(1.0)
    .onComplete { completed in
        if completed {
            onAnimationComplete()
        }
    }
```

### 用戶體驗
- **自動播放**：應用啟動時自動播放動畫
- **跳過功能**：用戶可以點擊「跳過」按鈕
- **僅首次**：使用 UserDefaults 記錄已觀看狀態
- **流暢過渡**：動畫結束後自動進入性別選擇畫面

## 故障排除
如果動畫無法播放：
1. 檢查 JSON 文件格式是否正確
2. 確認 Lottie 依賴已正確添加
3. 檢查動畫文件是否在正確的 Bundle 中
4. 查看 Xcode 控制台是否有錯誤訊息
